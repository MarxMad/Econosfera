# Valoración de Simuladores: FREE vs PRO vs RESEARCHER

Documento de análisis para definir qué simuladores ofrecer en cada plan según valor percibido, demanda y diferenciación.

---

## Criterios de valoración

| Criterio | Descripción |
|----------|-------------|
| **Demanda educativa** | ¿Lo usan estudiantes en cursos típicos? |
| **Demanda profesional** | ¿Lo usan analistas, consultores, investigadores? |
| **Complejidad** | Nivel técnico y sofisticación del modelo |
| **Diferenciación** | ¿Hay alternativas gratuitas (Excel, calculadoras online)? |
| **Costo de desarrollo** | Esfuerzo para mantener/mejorar |
| **Gancho comercial** | ¿Atrae conversión a PRO/RESEARCHER? |

## Propuesta de reasignación (cambios sugeridos)

### Subir a PRO (más valor premium)
- **Regresión múltiple (EViews)** — estadistica: FREE → PRO

### Bajar a FREE (más valor al plan gratuito)
- **Mult. transferencias** — macro: PRO → FREE
- **Estadísticas descriptivas** — estadistica: PRO → FREE
- **Break-even** — finanzas: PRO → FREE
- **Portafolio 2 activos** — finanzas: PRO → FREE
- **Trading** — blockchain: PRO → FREE
- **Staking** — blockchain: PRO → FREE

### Subir a RESEARCHER (más exclusividad)
- **AMM** — blockchain: ya RESEARCHER ✓
- **Regla de Taylor** — monetaria: podría quedarse PRO (es gancho fuerte)

## Notas finales

1. **FREE debe ser generoso** para el gancho: que el usuario vea valor y quiera más.
2. **PRO** debe incluir los simuladores que estudiantes avanzados y profesionales usan más.
3. **RESEARCHER** = diferenciación clara: IA, modelos avanzados, blockchain técnico.
4. Revisar periódicamente según métricas de uso y conversión.
